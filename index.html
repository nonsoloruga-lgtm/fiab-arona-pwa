FIAB Arona
Felici in Bici · Utility
Una sola app, tutto a portata di mano.
Mappe, link utili e colonnine ricaricabili. Funziona anche offline (per i contenuti già aperti).
Mappe
Apri mappe esterne o embed (QGIS/qgis2web, Google MyMaps, uMap, ecc.).
Nessuna mappa configurata.
Vai su “Impostazioni” e aggiungi i link/iframe.
Link utili
Raccolta rapida di risorse FIAB e ciclovie.
Nessun link configurato.
Vai su “Impostazioni” e aggiungi i link.
Colonnine ricaricabili
Inserisci punti di ricarica (e-bike / power bank / prese) e condividi l’elenco.
Nessun punto inserito.
Compila il form qui sopra per aggiungere il primo.
Impostazioni
Personalizza cosa mostrare nell’app. I dati restano sul dispositivo (localStorage).
Mappe (JSON)
Inserisci un array di oggetti: { "title": "...", "url": "...", "description": "...", "embedUrl": "..." }. embedUrl è opzionale (iframe).
Link (JSON)
Inserisci un array di oggetti: { "title": "...", "url": "...", "group": "FIAB" }.
Dati (backup)
Esporta/Importa tutte le impostazioni e colonnine in un file JSON.
